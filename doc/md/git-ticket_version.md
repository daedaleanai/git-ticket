## git-ticket version

Show git-ticket version information.

### Synopsis

Show git-ticket version information.

```
git-ticket version [flags]
```

### Options

```
  -n, --number   Only show the version number
  -c, --commit   Only show the commit hash
  -a, --all      Show all version information
  -h, --help     help for version
```
