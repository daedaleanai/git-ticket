## git-ticket config set

Set the named configuration data.

### Synopsis

Set the named configuration data.

```
git-ticket config set CONFIG [flags]
```

### Options

```
  -F, --file string   Take the config from the given file. Use - to read the config from the standard input
  -h, --help          help for set
```
