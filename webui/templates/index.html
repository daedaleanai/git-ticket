To add / edit entries in this sidebar update the "git-ticket" key in the local storage via the DeveloperConsole. The "git-ticket" key should contain a JSON object with a "bookmarks" property. Each property of the "bookmarks" object defines a group of bookmarks. Each bookmark group is a JSON object, where the key is the name of the bookmark and the value is a git ticket query (i.e., anything that can be passed to "git ticket ls").
